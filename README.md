# Stillpoint

A minimalist journaling website with separate journals and local browser storage.

## Files

- `index.html` - page structure
- `styles.css` - visual design
- `app.js` - journal behavior and local storage

## Publish with GitHub Pages

1. Open this repository on GitHub.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)` folder.
5. Save. GitHub will create a public site URL.

Entries are stored in the visitor's browser using local storage. They do not sync between devices yet.
